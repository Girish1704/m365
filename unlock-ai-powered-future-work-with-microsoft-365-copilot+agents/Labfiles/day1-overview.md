# Day 1: Unlock Productivity with Microsoft 365 Copilot Agents

## Estimated Duration: 4 Hours

## Overview

Day 1 focuses on exploring and utilizing pre-built agents available in Microsoft 365 Copilot. You will learn how to leverage specialized agents like Analyst, Researcher, and Document Analyzer to enhance productivity across different business scenarios. Additionally, you will create custom agents in Microsoft 365 Copilot and build a SharePoint agent to deliver contextual answers.

Through hands-on exercises, you will experience how these agents can transform everyday tasks, from data analysis to document processing, and how to automate knowledge assistance for your organization.

## Day 1 Topics Covered

- Microsoft 365 Copilot Overview
- Introduction to Microsoft 365 Copilot Agents
- Pre-built Agents in Microsoft 365
- Analyst Agent for Data Storytelling
- Researcher Agent for Content Organization
- Document Analyzer Agent for Word
- Creating Custom Copilot Agents
- SharePoint Agents for Contextual Answers

## Learning Objectives

After completing Day 1, you will be able to:

- Understand the capabilities of pre-built agents in Microsoft 365 Copilot
- Use the Analyst Agent to analyze data and create compelling visualizations
- Leverage the Researcher Agent to gather insights and organize content
- Utilize the Document Analyzer Agent to process and transform documents
- Create and configure custom Copilot Agents for knowledge assistance
- Build SharePoint agents that provide contextual answers from organizational content
