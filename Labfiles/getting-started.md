# Deploy and Adopt Microsoft 365 Copilot + Agents

## Overall Estimated Duration: 3 Days (4 Hours/Day)

## Overview

Microsoft 365 Copilot is an AI-powered productivity tool that uses large language models (LLMs) and integrates your data with the Microsoft Graph and Microsoft 365 Apps. In this hands-on lab, you will learn how Copilot provides real-time intelligent assistance, enabling users to enhance their creativity, productivity, and skills.

You will learn about technical requirements to access and configure Copilot in your organization and how you can extend it using plugins and agents. This lab covers everything from basic Copilot Chat interactions to building custom agents with Microsoft Copilot Studio and the Microsoft 365 Agents Toolkit.

## Objectives

By the end of this lab, you will be able to:

- Use Microsoft 365 Copilot Chat for event planning and content creation
- Create and configure agents in Copilot Chat
- Leverage pre-built agents to enhance employee productivity
- Empower sales teams with Microsoft 365 Copilot capabilities
- Create collaborative content using Copilot Pages
- Build custom agents using Microsoft Copilot Studio
- Develop declarative agents using Microsoft 365 Agents Toolkit
- Create SharePoint-based agents for organizational guidance
- Build autonomous agents and implement agent flows
- Implement prompt actions for dynamic content generation

## Pre-requisites

- Basic understanding of Microsoft 365 applications (Word, Excel, PowerPoint, Outlook, Teams)
- Familiarity with SharePoint and Microsoft Teams
- Basic knowledge of AI concepts and natural language processing
- Understanding of business workflows and automation concepts

## Architecture

The lab environment consists of the following components:

- **Microsoft 365 Tenant** — Pre-configured with Microsoft 365 Copilot licenses
- **Microsoft Copilot Studio** — For building custom agents and flows
- **Microsoft 365 Agents Toolkit** — VS Code extension for declarative agent development
- **SharePoint Online** — For creating SharePoint-based agents
- **Microsoft Teams** — For testing and deploying agents
