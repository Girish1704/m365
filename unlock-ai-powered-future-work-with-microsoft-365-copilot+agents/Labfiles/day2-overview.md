# Day 2: Extend Microsoft 365 Copilot with Custom Agents

## Estimated Duration: 4 Hours

## Overview

Day 2 focuses on extending Microsoft 365 Copilot capabilities by building custom agents using Microsoft Copilot Studio and the Microsoft 365 Agents Toolkit. You will learn how to create specialized agents that integrate with your organizational data and provide domain-specific assistance.

Through hands-on exercises, you will build an HR agent using Copilot Studio and create a declarative agent using the Agents Toolkit in Visual Studio Code. These skills will enable you to customize Copilot for your organization's unique needs.

## Day 2 Topics Covered

- Extend Microsoft 365 Copilot with Agents
- Agent Options for Microsoft 365
- Microsoft Copilot Studio Overview
- Building Agents with Copilot Studio
- Declarative Agents Introduction
- Microsoft 365 Agents Toolkit
- Agent Configuration and Customization
- Publishing and Testing Agents

## Learning Objectives

After completing Day 2, you will be able to:

- Build custom agents using Microsoft Copilot Studio
- Configure agent topics, triggers, and responses
- Connect agents to knowledge sources
- Create declarative agents using the Microsoft 365 Agents Toolkit
- Define agent behavior using JSON manifests
- Test and deploy agents for organizational use
