# Exercise 3: Enhance Employee Productivity with Pre-built Agents

## Estimated Duration: 45 Minutes

## Overview

In this exercise, you will explore and utilize pre-built agents available in Microsoft 365 Copilot. Pre-built agents are ready-to-use AI assistants designed for common business scenarios. These agents come with predefined instructions, knowledge sources, and capabilities that can be immediately deployed to enhance employee productivity.

You will discover various pre-built agents, learn how to access and use them, and understand how they can be customized or serve as templates for your own agents.

## Exercise Objectives

In this exercise, you will complete the following tasks:

- Task 1: Explore the Pre-built Agents Gallery
- Task 2: Use the Writing Coach Agent
- Task 3: Use the Project Manager Agent
- Task 4: Use the Researcher Agent
- Task 5: Compare and evaluate agent capabilities

### Task 1: Explore the Pre-built Agents Gallery

In this task, you will navigate to the pre-built agents gallery and explore available options.

1. In the Microsoft 365 Copilot Chat interface, click on the **Copilot agents** or **Agents** icon in the navigation.

   ![](./media/ex3-agents-navigation.png)

1. In the agents panel, look for the **Gallery**, **Browse**, or **Discover** section that contains pre-built agents.

   ![](./media/ex3-agents-gallery.png)

1. You will see various categories of pre-built agents, such as:

   - **Productivity** — Writing, scheduling, task management
   - **Communication** — Email drafting, meeting preparation
   - **Research** — Information gathering, summarization
   - **Creative** — Brainstorming, content creation
   - **Business** — Sales, HR, Finance assistance

   ![](./media/ex3-agent-categories.png)

1. Browse through the available agents and note the following for each:
   - Agent name and description
   - Key capabilities
   - Sample prompts or use cases

   ![](./media/ex3-agent-details.png)

   >**Note:** The availability of pre-built agents may vary based on your organization's Microsoft 365 subscription and tenant configuration.

### Task 2: Use the Writing Coach Agent

In this task, you will use a writing-focused pre-built agent to improve your professional communication.

1. From the agents gallery, select the **Writing Coach** or similar writing-assistance agent.

   ![](./media/ex3-writing-coach.png)

1. Click **Start chat** or **Use this agent** to begin a conversation.

   ![](./media/ex3-start-writing-coach.png)

1. Test the Writing Coach with the following prompt to improve an email:

   ```
   Please help me improve this email:

   "Hey team, 
   
   just wanted to let u know that the meeting is moved to tomorrow. sorry for the short notice but something came up. pls confirm if u can make it.
   
   thanks"
   ```

   ![](./media/ex3-writing-coach-prompt.png)

1. Review the improved version provided by the Writing Coach. It should offer a more professional tone while maintaining the original message.

   ![](./media/ex3-writing-coach-response.png)

1. Ask for alternative versions:

   ```
   Can you provide a more formal version suitable for sending to senior leadership?
   ```

   ![](./media/ex3-formal-version.png)

1. Request specific improvements:

   ```
   Now help me write a compelling subject line for this rescheduling email.
   ```

   ![](./media/ex3-subject-line.png)

1. Test the Writing Coach for different content types:

   ```
   I need to write a LinkedIn post announcing our team's achievement of exceeding Q4 targets by 25%. It should be professional but celebratory, around 150 words.
   ```

   ![](./media/ex3-linkedin-post.png)

### Task 3: Use the Project Manager Agent

In this task, you will explore a project management-focused agent to help organize work.

1. Navigate back to the agents gallery and select the **Project Manager**, **Task Planner**, or similar organizational agent.

   ![](./media/ex3-project-manager.png)

1. Start a conversation with the Project Manager agent.

1. Ask for help planning a project:

   ```
   I'm starting a new website redesign project with a 3-month timeline. The team includes 2 designers, 3 developers, and 1 content writer. Can you help me create a project plan with major milestones?
   ```

   ![](./media/ex3-pm-project-plan.png)

1. Review the project plan and ask for more details:

   ```
   This is helpful. Can you break down the first milestone "Discovery and Planning" into specific tasks with estimated durations and team member assignments?
   ```

   ![](./media/ex3-pm-tasks.png)

1. Ask for risk identification:

   ```
   What are the potential risks for this website redesign project, and how should we mitigate them?
   ```

   ![](./media/ex3-pm-risks.png)

1. Request a status report template:

   ```
   Create a weekly status report template I can use to track progress on this project.
   ```

   ![](./media/ex3-pm-status-template.png)

### Task 4: Use the Researcher Agent

In this task, you will use a research-focused agent to gather and synthesize information.

1. Navigate to the agents gallery and select the **Researcher**, **Analyst**, or similar information-gathering agent.

   ![](./media/ex3-researcher.png)

1. Start a conversation and request research assistance:

   ```
   I need to understand the current trends in AI adoption for small and medium businesses. Can you provide a summary of key trends, benefits, and challenges?
   ```

   ![](./media/ex3-researcher-prompt.png)

1. Review the research summary and ask for deeper analysis:

   ```
   Can you elaborate on the implementation challenges and provide recommendations for overcoming them?
   ```

   ![](./media/ex3-researcher-challenges.png)

1. Request competitive analysis:

   ```
   Summarize how competitors in the retail industry are using AI for customer experience improvement. Provide 3-4 examples with key takeaways.
   ```

   ![](./media/ex3-researcher-competitive.png)

1. Ask for a structured report:

   ```
   Compile everything we've discussed into an executive summary format with sections for: Overview, Key Trends, Challenges, Recommendations, and Next Steps.
   ```

   ![](./media/ex3-researcher-summary.png)

### Task 5: Compare and Evaluate Agent Capabilities

In this task, you will compare the different pre-built agents and understand their strengths.

1. Create a comparison by testing the same prompt across different agents. First, ask the Writing Coach:

   ```
   Help me communicate our department's Q1 achievements to stakeholders.
   ```

   ![](./media/ex3-compare-writing.png)

1. Switch to the Project Manager agent and ask the same question:

   ```
   Help me communicate our department's Q1 achievements to stakeholders.
   ```

   ![](./media/ex3-compare-pm.png)

1. Ask the Researcher agent the same:

   ```
   Help me communicate our department's Q1 achievements to stakeholders.
   ```

   ![](./media/ex3-compare-researcher.png)

1. Note the differences in how each agent approaches the same request:

   | Agent | Approach | Best For |
   |-------|----------|----------|
   | Writing Coach | Focuses on language, tone, and presentation | Polishing communication |
   | Project Manager | Focuses on structure, timelines, deliverables | Organizing information |
   | Researcher | Focuses on data gathering and analysis | Supporting with evidence |

   ![](./media/ex3-comparison-table.png)

1. Based on your exploration, identify which agents would be most valuable for your daily work:

   ```
   Based on our conversation, which pre-built agents would you recommend for someone in a marketing role?
   ```

   ![](./media/ex3-recommendations.png)

   >**Note:** Pre-built agents serve as excellent starting points. You can use them as-is for common scenarios or as templates to create customized versions that better fit your organization's specific needs.
