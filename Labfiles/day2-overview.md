# Day 2: Prepare Your Organization for Microsoft 365 Copilot

## Estimated Duration: 4 Hours

## Overview

Day 2 focuses on extending Microsoft 365 Copilot capabilities by building custom agents using Microsoft Copilot Studio and the Microsoft 365 Agents Toolkit. You will learn how to create specialized agents that integrate with your organizational data and provide domain-specific assistance.

Through hands-on exercises, you will build an HR agent using Copilot Studio, create a declarative agent using the Agents Toolkit in Visual Studio Code, and develop a SharePoint-based agent for career guidance. These skills will enable you to customize Copilot for your organization's unique needs.

## Day 2 Topics Covered

- Extend Microsoft 365 Copilot with Agents
- Agent Options for Microsoft 365
- Extend Copilot by building Declarative agents
- Choosing the right tools to build your agents
- Build agents with Copilot Studio Agent builder
- Build SharePoint Agents
- Building declarative agents with Copilot Studio
- Add skills with Copilot Studio Actions
- Publish your agents

## Learning Objectives

After completing Day 2, you will be able to:

- Build custom agents using Microsoft Copilot Studio
- Understand the different agent building approaches and when to use each
- Create declarative agents using the Microsoft 365 Agents Toolkit
- Develop SharePoint-based agents for organizational knowledge
- Configure agent topics, triggers, and responses
- Connect agents to external data sources and APIs
- Test and publish agents for organizational use
